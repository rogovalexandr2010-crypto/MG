MineGame — обновлённый пакет веб-версии
======================================

Что внутри:
- index.html — интерфейс (тёмная тема, сохранение в localStorage)
- style.css — стили (темная минималистичная палитра)
- game.js — логика игры: руды, кирки, добыча, магазин, сохранение

Как использовать:
1. Распакуй файлы на свой хостинг или локальную папку.
2. index.html — главный файл для открытия (Web App будет открывать эту страницу).
3. Вставь файлы на render/Netlify/Vercel или любой простой static-хост.
4. Бот Telegram у тебя остаётся как есть — он открывает WEBAPP_URL, укажи URL на этот index.html.

Коротко про механику:
- Есть 5 видов руды: Камень, Уголь, Железо, Золото, Алмаз.
- Есть 5 кирок — повышение силы увеличивает число попыток добычи и доход.
- Баланс и инвентарь сохраняются в localStorage автоматически.

Если хочешь, могу:
- добавить кнопки продажи руды (продажа всего или частично),
