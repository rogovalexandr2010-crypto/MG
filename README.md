MineGame v2 - автономный Web App
=====================================

Файлы:
- index.html — интерфейс игры
- style.css — стили (темная тема)
- game.js — логика игры: добыча, руды, кирки, продажа
- README.md — инструкция

Особенности:
- 5 видов руды: Камень, Уголь, Железо, Золото, Алмаз
- 5 кирок с разной силой и ценой
- Кнопка "Добыть" только добавляет руду
- Кнопка "Продать всё" начисляет деньги
- Прогресс сохраняется автоматически через localStorage

Использование:
- Залей файлы на GitHub Pages или любой static-хост
- Обнови WEBAPP_URL в Telegram-боте на URL index.html
